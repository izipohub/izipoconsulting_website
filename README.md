# Izipo Consulting - Company Website

A modern, interactive company website for Izipo Consulting, featuring a stunning 3D island experience, comprehensive service pages, and a professional portfolio showcase.

![Izipo Consulting](https://izipoconsulting.com/og-image.jpg)

## 🚀 Live Demo

Visit the live site: [https://izipoconsulting.com](https://izipoconsulting.com)

## ✨ Features

### 🌟 Interactive 3D Island
- Immersive 3D island experience powered by Three.js and React Three Fiber
- Rotatable island with interactive hotspots that reveal information boxes
- Animated birds, plane, and sky for a living, breathing environment
- Background music with toggle control
- Floating information cards that appear based on island rotation

### 📱 Responsive Design
- Fully responsive across all devices (mobile, tablet, desktop)
- Mobile-optimized navigation with hamburger menu
- Adaptive 3D model scaling for different screen sizes

### 📄 Comprehensive Pages
- **Home**: 3D island hero with rotating information panels
- **About**: Company story, mission, values, and expertise
- **Services**: Detailed service offerings with interactive cards
- **Projects**: Portfolio showcase with filtering and project modals
- **Contact**: Contact form with 3D fox mascot animation

### 🎨 Design Features
- Dark theme with glass morphism effects
- Gradient accents and smooth animations
- Custom SVG icons for social media
- Animated counters for statistics
- Hover effects and transitions throughout
